⚙ Segment Init
DS ← @DATA MOV AX, @DATA
MOV DS, AX
Can't load DS directly;
AX used as intermediary
@DATA is the segment address of .DATA section.
8086 cannot MOV directly into DS, so AX is used as a relay.
After this, [DX] memory references resolve into .DATA.
⚡ DOS Func Select
AH=09h → Display String MOV AH, 09H
LEA DX, MSG
LEA loads effective address
of MSG prompt into DX
AH=09h tells INT 21h to print a '$'-terminated string.
LEA DX, MSG loads the offset address of the MSG variable
into DX so INT 21h knows where to start printing.
📺 Display Prompt
INT 21h / AH=09h INT 21H
Output ▶ ENTER UPPERCASE LETTER:
DOS Interrupt 21h with AH=09h reads the string pointed to by DS:DX
and prints each character until it hits the '$' terminator.
Output: ENTER UPPERCASE LETTER:
⚡ DOS Func Select
AH=01h → Read Char MOV AH, 01H
Selects keyboard read
with echo function
AH=01h prepares INT 21h to wait for a single keypress.
The character is echoed to screen and stored in AL register.
⌨ Read Character
INT 21h / AH=01h INT 21H
Waits for keypress
 AL ← ASCII of key pressed
INT 21h pauses execution, waiting for a keystroke.
The ASCII value of the typed character is stored in AL.
Only uppercase A–Z should be entered per program design.
⇄ Register Transfer
AL → DL MOV DL, AL
Copy char from input reg
to output reg
AL holds the character read by INT 21h/01h.
DL is required by INT 21h/02h to output a character.
This MOV bridges input and output services.
📺 Display Char
AH=02h + INT 21h MOV AH, 02H
INT 21H
Prints char in DL to screen
AH=02h selects the "Display Character" DOS function.
INT 21h then outputs the character in DL to stdout.
The uppercase letter typed by the user is echoed again.
✕ Terminate
AH=4Ch + INT 21h MOV AH, 4CH
INT 21H
Return control to DOS
Exit code in AL (0 = OK)
AH=4Ch is the DOS "Terminate with Return Code" function.
AL contains the exit code (default 0 = success).
Cleanly returns control to the DOS command prompt.
